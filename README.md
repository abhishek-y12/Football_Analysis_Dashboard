# Football_Analysis_Dashboard
Project Overview

This project is a Football Analysis Dashboard created using Power BI. It helps analyze football match data and understand how teams perform in attack and defence. The dashboard uses different charts and filters so users can easily explore team statistics.

Tools Used -
Power BI,
SQL,
Exce,l
DAX,
Data Cleaning

Home Page -
This page shows overall match insights. It includes points gained from losing positions and points lost from winning positions. Users can also compare two teams using the head-to-head chart. It also displays match statistics like average possession, total corners, offsides, and free kicks.

Defence Analysis Page -
This page focuses on the defensive performance of teams. It shows statistics like clean sheets, saves, save percentage, fouls per game, yellow cards, red cards, and goals conceded. It also shows how teams respond to pressure during matches.

Attack Analysis Page -
This page analyzes the attacking performance of teams. It includes home goals, away goals, shot accuracy, and conversion rate. It also shows which teams score the most goals and how goals are distributed between the first half and second half
